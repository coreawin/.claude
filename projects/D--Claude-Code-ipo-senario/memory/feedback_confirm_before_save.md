---
name: 파일 저장 전 사용자 확인 필수
description: 결과물 파일을 저장하기 전에 반드시 사용자에게 내용을 먼저 보여주고 확인을 받아야 함
type: feedback
---

결과물 파일을 작성할 때는 반드시 내용을 먼저 채팅으로 보여주고 사용자 확인 후에 저장한다.

**Why:** 사용자가 검토·수정 없이 파일이 바로 저장되는 것을 원하지 않음. 확인 작업을 거친 후 저장이 원칙.

**How to apply:** 새 문서나 스크립트를 생성할 때, Write 도구 사용 전에 항상 채팅으로 초안을 출력하고 "확인되시면 저장하겠습니다" 흐름으로 진행.
