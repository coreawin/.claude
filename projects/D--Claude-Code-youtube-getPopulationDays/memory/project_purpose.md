현재 인기있거나 트렌드가 있는 정보성 콘텐츠를 찾아 분석하고, 유사한 AI 영상을 빠르게 생성해 업로드하기 위한 정보 수집이 목적.

**Why:** 트렌드 선점이 핵심 — 효율성(구독자 대비 조회수)이 높은 영상 = 지금 뜨고 있는 주제 신호

**How to apply:**
- AI 분석 기능은 "이 영상과 유사한 스크립트/주제 생성"에 초점을 맞춰 설계
- 검색 필터는 재현 가능한 정보성 콘텐츠(교육, 과학, 역사, 뉴스) 중심으로 최적화
- 스포츠, 음악, 게임, 영화/애니 등 AI로 재현하기 어려운 카테고리는 기본 제외
- 기본 기간은 1주일 (오늘은 조회수/댓글 미축적으로 품질 필터 탈락 多)
- 기본 영상 길이는 4~20분 (정보성 콘텐츠 주요 구간, 쇼츠 제외)
