## 구현 계획

### 파일 구조
```
20260327/
├── qa_result.png (기존)
└── output/
    ├── index.html
    ├── style.css
    └── script.js
```

### index.html 구조
```html
<div class="container">
  <div class="panel left-panel">
    <!-- 콜 상담 정보 테이블 -->
    <!-- 대화 상세 (채팅 말풍선) -->
  </div>
  <div class="panel right-panel">
    <!-- 체크리스트 카테고리별 섹션 -->
    <!-- 하단 총점 -->
  </div>
</div>
```

### CSS 핵심 스타일
- 좌우 패널: `display: flex`, 50:50 분할
- 채팅 말풍선: flex + justify-content로 좌/우 배치
- 체크리스트 카테고리 헤더: 색상 배경 (보라 `#7c5cbf`, 파랑 `#4a7fc1`, 주황 `#d97706`)
- 전체 배경: 연한 회색 (`#f5f5f5`)

### JavaScript 핵심 기능
- 체크박스 클릭 시 총점 자동 재계산
- "대화 원문 보기" 버튼 클릭 시 토글 (선택적)
- 카테고리 접기/펼치기 (선택적)
